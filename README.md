# ALRAMZ0

مشروع منصة تحليل رضا العملاء وتوفير رؤى استراتيجية لتحسين تجربة المستخدم.

## الميزات الرئيسية

- لوحة تحكم تفاعلية لعرض مؤشرات الأداء الرئيسية
- تحليل بيانات رضا العملاء
- تقارير وإحصائيات مفصلة
- واجهة مستخدم حديثة وسهلة الاستخدام

## التقنيات المستخدمة

- React
- TypeScript
- Tailwind CSS
- Vite
